## Phase Details

### Phase 10: Jobs Awareness
**Goal**: Agents know that jobs exist and auto-receive context when they read a JOB.md
**Depends on**: Phase 9 (outside GSD tracking)
**Requirements**: JOBS-01, JOBS-02, JOBS-03, JOBS-04, JOBS-05
**Success Criteria** (what must be TRUE):
  1. Agent system prompt contains a Jobs section with the jobs directory path — no job listing
  2. When an agent reads a JOB.md, its MEMORY.md appears in agent history without an explicit read call
  3. When an agent reads a JOB.md with skill-deps, those skills' SKILL.md and MEMORY.md appear in agent history without explicit reads
  4. A SKILL.md read cannot surface job body content (isolation boundary preserved)
  5. All existing skill-memory tests pass and new behavioral tests cover criteria 2, 3, and 4 above
**Plans**: 1 plan
